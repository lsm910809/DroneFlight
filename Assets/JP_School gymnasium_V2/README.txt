Description:
--------------------
With this package you can make a "Japanese school gymnasium".

This package contains:

- 160 meshs. (Polygon count : Min 2 - Max 10980)
- 256 prefabs.
- 141 materials.
- 311 textures. (1024x1024/2048x2048)
- 1 skybox procedural material.
- 1 demo scene.
- 3 post processing stack v2 profiles.

Been used for AssetStore screenshots visualization:

- Post Processing Stack V2
